# HR-Analytics-Dashboard
Project Overview
This HR Analytics Dashboard is designed to analyze employee data and provide
actionable insights for management using interactive visuals.

Tools Used
- Microsoft Advance Excel
- Power BI

Key Analysis
- Employee Attrition Analysis
- Department-wise Headcount
- Gender Diversity Analysis
- Performance & Attendance Overview
- Workforce Trends

 Files in this Repository
- HR Analytics Dashboard Overview.png (Dashboard snapshot)
- Power BI Dashboard.mp4 (Dashboard walkthrough video)

 Objective
To help HR and management teams make data-driven decisions
using clear and interactive dashboards.
